Tiles + Cards

To use Tiles and Cards you'll need to adjust the class based on how many you want to display. Currently, it accepts 1-4 tiles. To update this, just alter the parent element that contains the Cards to have the class: tiles count-3

Media : Collage

Images have multiple settings. Lets start with Text, you can add a Title and/or a Description. You can also add the class fade-in-title to the anchor tag if you want to show the image by default and fade both the Title and Description in.

Lastly, there are additional settings to control aspect ratio but this was set up in case we ever need a Gallery of some type.
